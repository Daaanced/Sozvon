Назначение Authorization Service

Auth Service — отдельный микросервис, который занимается:

Проверкой личности пользователей (логин/пароль)

Выдачей токенов (JWT) для доступа к другим сервисам

Валидацией токенов при обращении к Gateway

Опционально:

сброс пароля

хранение ролей/прав

логирование входов

Важно: Auth Service не хранит профили пользователей. Все данные о пользователях (имя, аватар, адрес) — в User Service.

auth_service/          ← корневая папка сервиса
├── main.go            ← точка входа, запускает сервер
├── db.go              ← работа с PostgreSQL
├── auth.go            ← JWT, хеширование паролей
├── handlers.go        ← обработчики HTTP
├── models.go          ← структуры данных (User)
└── go.mod             ← модуль Go
